# Code of Conduct

WevoaWeb is an open-source project and we want it to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Give feedback about code and ideas, not about people.
- Help keep discussions technical, calm, and collaborative.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination
- Personal attacks or repeated hostile behavior
- Deliberate disruption of discussions or contributions
- Sharing private information without permission

## Maintainer Responsibility

Project maintainers may moderate discussions, edit or remove inappropriate content, or restrict participation when necessary to protect the community.

## Reporting

If you experience or witness unacceptable behavior, please report it privately through the security or maintainer contact channel described in [SECURITY.md](SECURITY.md).
