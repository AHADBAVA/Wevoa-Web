# Security Policy

## Supported Status

WevoaWeb is currently experimental. Security improvements are welcome, but no long-term support guarantees are currently offered.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive problems.

Instead, report:

- the affected component
- the impact
- steps to reproduce
- any suggested fix if available

If you are the repository owner, add your preferred security contact on GitHub after the first push. Until then, please coordinate privately with the maintainers.

## Response Goals

The project will try to:

- acknowledge reports promptly
- confirm reproducibility
- prepare a fix or mitigation
- document any important user-facing guidance
