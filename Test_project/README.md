# Test_project

This is the bundled sample WevoaWeb application included with the framework repository.

## Structure

- `app/` backend routes and SQLite bootstrapping
- `views/` templates and layout files
- `public/` static assets
- `storage/` SQLite database files
- `wevoa.config.json` app configuration

## Run

From the repository root:

```powershell
cd .\Test_project
..\wevoa.exe start
```

Then open:

```text
http://localhost:3000
```
