Thanks for using WevoaWeb.

If you need help with the project:

- read the [README](README.md)
- check the [Quick Start guide](docs/quickstart.md)

- check the [Framework Overview](docs/framework-overview.md)

## When to Open an Issue

Open an issue when you have:

- a reproducible bug
- a clear feature request
- a documentation problem

Please include:

- your operating system
- compiler version
- the runtime version from `wevoa --version`
- the command you ran
- the `.wev` source or steps needed to reproduce the problem

## What Support Looks Like

WevoaWeb is maintained as a compact evolving platform.

That means support is currently best-effort and is strongest for:

- reproducible bugs
- documentation fixes
- installer/runtime problems
- clearly scoped feature gaps

## Before Reporting a Runtime Bug

Please check:

- are you using the latest runtime version
- are you running the intended binary
- does the issue reproduce in a fresh generated starter project
- does `wevoa doctor` report any project problems

## Security

If the issue is security-sensitive, do not open a public issue.

Please follow the guidance in [SECURITY.md](SECURITY.md).
